# Asset Management 
> 包含如下子模块:
> * Common
> * AssetBuilder
> * AssetBrowser
> * Runtime
> * GameBuilder

## Common
* 全局配置及工具类

## Asset Builder
* 设置AssetBundle Names
* 检查资源名称合法性
* 资源导入后处理（AssetPostprocessor）

## Asset Browser
* 资源检查工具：AB的依赖数量、是否依赖内置或外部资源

## Runtime
* AssetManager： 运行时资源加载接口
* 可视化工具：运行时的资源引用

## Game Builder
